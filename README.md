# HTML → PNG 変換ツール

HTMLコードを貼り付けるか、.htmlファイルをアップロードしてPNG画像に変換できるWebツールです。

## 使い方

1. index.html をブラウザで開く
2. HTMLコードをテキストエリアに貼り付け、または .html ファイルをアップロード
3. プレビューで確認（幅・高さを調整可能）
4. 「PNGに変換してダウンロード」ボタンをクリック

## GitHub Pages へのデプロイ方法

1. GitHubに新しいリポジトリを作成
2. このフォルダの内容をプッシュ
3. リポジトリの Settings → Pages → Source を main ブランチに設定
4. 数秒後に https://<ユーザー名>.github.io/<リポジトリ名>/ でアクセス可能

## 注意事項

- 外部フォント・外部画像（別ドメイン）を含むHTMLは正しく変換されない場合があります
- 変換はブラウザ内で完結します（サーバーへのデータ送信はありません）
